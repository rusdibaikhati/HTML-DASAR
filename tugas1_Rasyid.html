Apa Itu HTML

HTML itu adalah sebuah bahasa yang menggunakan markup atau penada untuk pembuat halaman web. penada atau markup ini, nanti akan kita sebut tag penanda atau markup ini, nanti akan kita sebut dengan HTML berperan untuk menemukan struktur kontendan tampilan dari sebuah web.kalau kita ibaratkan nih. HTML itu seperti batu bta untuk membangun rumah. Batu bata ini dapat disusun, seperti memiliki sumber komunitas yang cukup besar dengan pengguna yang luas. tidak hanya, kelebihan lainnya adalah HTML mudah untuk dipelajari oleh seorang web developer pemula.kelebihan adalah HTML merupakan bahasa pemrogramanyang gratis atau open source.
